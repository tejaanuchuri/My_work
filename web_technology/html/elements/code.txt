1. name : code
2. description : The HTML <code> element displays its contents styled in a fashion intended to indicate that the text is a short fragment of computer code. By default, the content text is displayed using the user agent's default monospace font.
3. example :
    <p>The <code>push()</code> method adds one or more elements to the end of an array and returns the new length of the array.</p>

    display:
    The push() method adds one or more elements to the end of an array and returns the new length of the array.
